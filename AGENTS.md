# CookIt

## Guidelines

- We follow spec driven development (SDD) and use superpowers.
- Use `Bun` for everything.
- Use `oxfmt` and `oxlint` for code formatting and linting, never prettier or eslint.
- Use `cli` tools whenever possible for create projects and manage dependencies.
- All env vars should be places in root `.env` file.
- The final project should be run with a docker compose command.
- UI should use `ShadcnUI` components library with `Tailwindcss`.
